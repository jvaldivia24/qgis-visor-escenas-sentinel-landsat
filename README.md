# Visor de escenas Sentinel-2 y Landsat – SERNANP

Complemento para QGIS que permite la exploración, previsualización y carga de escenas satelitales Sentinel-2 y Landsat (5, 7, 8 y 9), orientado a flujos de trabajo de análisis y monitoreo ambiental.

## Funcionalidades
- Búsqueda de escenas por ubicación y rango de fechas
- Previsualización directa de escenas dentro del complemento mediante:
  - Color natural
  - Infrarrojo
  - Agricultura
- Carga inmediata de escenas al visor de QGIS
- Generación de índices espectrales NDVI y NBR a partir de la escena seleccionada
- Visualización optimizada mediante recortes espaciales para mejorar el rendimiento

## Requisitos
- QGIS versión 3.22 o superior

## Uso principal
Este complemento está orientado a profesionales del Servicio de Áreas Naturales Protegidas (SERNANP) y publico en general que trabajan en:
- Monitoreo ambiental
- Gestión territorial
- Conservación de ecosistemas
- Análisis de cobertura y cambios del uso del suelo
